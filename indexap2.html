List of software developed

The following is a list of models, software tools and scripts developed for this thesis and the locations where the source code can be found.

  • A collection of cell models and tools for model building, simulation and analysis will be made available at https://github.com/lkoelman/bg_models_tools
  • The code for Chapter 3 will be made available at https://github.com/lkoelman/stn-gpe_model
  • The code for Chapter ?? will be made available at https://github.com/lkoelman/stn-gpe_dbs_model
  • The code for Chapter 5 will be made available at https://github.com/lkoelman/stn-gpe_reduced_model.

List of software used

The following software was used for the implementation and analysis of neuronal network and cell models in this thesis:

  • The NEURON simulation environment by [153] (https://neuron.yale.edu/neuron/)
  • PyNN: a Python package for simulator-independent specification of neuronal network models by [73] (http://neuralensemble.org/PyNN/)
  • Blender, an open source 3D creation suite (https://www.blender.org/)
  • NeuroMorphoVis, a Blender add-on for building, visualizing and analyzing digital reconstructions of neuronal morphology skeletons by [4] (https://github.com/BlueBrain/NeuroMorphoVis)
